PCT-100 化工过程控制实训装置

基于 ESP32-C3 的智能环境监控与自动控制系统

📋 功能概览

🌡️

温度监测

DS18B20 数字传感器，精度 ±0.5℃，实时采集并显示

💡

光照监测

模拟光敏传感器 + 12-bit ADC，光照越亮数值越高

🔌

双路继电器

灯光 + 风扇独立控制，支持手动四态 / 自动联动两种模式

📺

OLED 显示

128×64 中文界面，实时显示温度、光照、继电器状态、网络信息

🌈

RGB 状态灯

WS2812 万色灯，6 种颜色/闪烁模式直观反馈设备状态

📶

WiFi + MQTT

2.4GHz WiFi 联网，MQTT 远程监控，支持内外网切换

🔘 按键操作

按键操作功能
KEY1按下 (自锁)总闸关闭：强制关闭所有继电器，进入自动模式
KEY1松开总闸开启：恢复运行，允许模式切换和继电器控制
KEY2短按 (< 0.8s)手动模式下，循环切换继电器四种状态 (00→01→10→11→00…)
KEY2长按 (≥ 2s)切换 自动 / 手动 模式

🤖 自动控制逻辑

自动模式下，传感器数据驱动继电器动作：

控制对象触发条件动作
灯光光照 ADC > 阈值 (1500)继电器1 吸合 → 灯亮
灯光光照 ADC < 阈值 (1000)继电器1 释放 → 灯灭
风扇温度 > 31.0℃继电器2 吸合 → 风扇转
风扇温度 < 30.0℃继电器2 释放 → 风扇停
💡 光照使用滞回控制（开灯 1500 / 关灯 1000），温度使用 1℃ 回差，防止继电器在临界值附近频繁抖动。
