# 硬件连接说明

## 主控芯片

**ESP32-C3** (RISC-V 32bit, 160MHz, 400KB SRAM, 4MB Flash)

## 引脚分配

| GPIO | 功能 | 连接器件 | 说明 |
|------|------|----------|------|
| **0** | RGB LED | WS2812 (U4) | 单总线，板载集成 |
| **1** | ADC 输入 | 光敏传感器 | 12-bit 精度，0–4095 |
| **4** | I²C SDA | SSD1306 OLED | 128×64，I²C 硬件 |
| **5** | I²C SCL | SSD1306 OLED | 128×64，I²C 硬件 |
| **6** | 数字输出 | 继电器 1（灯光） | 高电平吸合 |
| **7** | 数字输出 | 继电器 2（风扇） | 高电平吸合 |
| **10** | OneWire | DS18B20 温度传感器 | 单总线数字温度 |
| **20** | 数字输入 | KEY1（总闸） | 内部上拉，按下低电平 |
| **21** | 数字输入 | KEY2（模式） | 内部上拉，按下低电平 |

## 硬件框图

```
                    ┌─────────────────────────┐
                    │      ESP32-C3           │
                    │                         │
   KEY1 ───────────┤ GPIO20 (INPUT)           │
   KEY2 ───────────┤ GPIO21 (INPUT)           │
                    │                         │
   WS2812 ─────────┤ GPIO0  (OUTPUT)          │
   OLED  ──────────┤ GPIO4  (I²C SDA)        │
   OLED  ──────────┤ GPIO5  (I²C SCL)        │
   RELAY1(灯光) ───┤ GPIO6  (OUTPUT)          │
   RELAY2(风扇) ───┤ GPIO7  (OUTPUT)          │
   DS18B20 ────────┤ GPIO10 (OneWire)         │
   光敏传感器 ─────┤ GPIO1  (ADC 12-bit)      │
                    └─────────────────────────┘
```

## 传感器规格

### 光照传感器

- 类型：模拟量光敏电阻模块
- 接口：ADC (GPIO1)
- 分辨率：12-bit (0–4095)
- 对应电压：0–3.3V
- 光照越亮 → ADC 值越低（电阻减小）
- **Lux 换算**：`lux = (4095 - adc)² / 30000`
- 默认开灯阈值：1500 (ADC) / 200 (lux)
- 默认关灯阈值：1000 (ADC)

### 温度传感器

- 型号：DS18B20
- 接口：OneWire (GPIO10)
- 精度：±0.5℃
- 范围：-55℃ ~ +125℃
- 读数间隔：500ms
- 默认开风扇阈值：31.0℃
- 默认关风扇阈值：30.0℃（1℃ 回差）

### OLED 显示屏

- 驱动：SSD1306
- 分辨率：128×64
- 接口：I²C (GPIO4=SDA, GPIO5=SCL)
- 字库：U8g2 中文字库 (wqy12)
- 刷新间隔：1 秒

### RGB LED

- 型号：WS2812 (NeoPixel)
- 数量：1 颗（板载 U4）
- 接口：单总线 (GPIO0)
- 亮度：32/255 (节能，适合持续显示)

## 电源要求

- 主控供电：3.3V (板载 LDO)
- 继电器供电：5V (独立供电)
- 整机输入：5V DC (USB-C 或排针)
