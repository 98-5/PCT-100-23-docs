🎮 操作指南

从开机到远程控制，快速上手 PCT-100

🚀 快速开始

  1. 上电：接入 5V DC 电源，ESP32 自动启动
  2. 开机自检：RGB LED 彩虹闪烁两轮（约 1.2 秒），OLED 显示系统信息
  3. WiFi 连接：设备自动连接上次保存的 WiFi（首次使用需配网）
  4. MQTT 连接：WiFi 成功后自动连 MQTT 服务器，OLED 底部显示服务器地址
  5. 开始使用：默认自动模式，系统根据传感器数据自动控制灯光和风扇

📶 WiFi 配网流程

首次使用或更换网络时需要配网。已保存的网络会自动连接。

初级模式（代码写定 WiFi）

设备上电后自动连接代码中预设的 WiFi（SSID: 普通市民），无需手动操作。

升级模式（串口选网）

  1. 打开串口监视器（波特率 115200）
  2. 输入 s 并回车，设备扫描附近 WiFi 并列出
  3. 输入 网络编号 选择目标网络
  4. 输入 WiFi 密码（明文）
  5. 连接成功后自动保存到 Flash，下次开机自动连接

🔘 按键操作详解

KEY1 — 总闸（自锁按键）

状态效果
按下（🔴 关闭）强制关闭所有继电器，锁定为自动模式。
所有 MQTT 控制指令被忽略，OLED 总闸显示 OFF。
松开（🟢 开启）系统正常运行。自动模式下传感器自动控制，手动模式下可按键或远程控制。
WiFi 和 MQTT 正常通信。
⚠️ KEY1 是最高优先级的安全锁。当 KEY1 关闭时，PC 上位机的模式切换和继电器控制按钮都会被禁用。

KEY2 — 功能键（轻触按键）

操作时长效果
短按50ms ~ 800ms手动模式下循环切换继电器四态
00 (全关) → 01 (风扇开) → 10 (灯开) → 11 (全开) → 00…
长按≥ 2000ms切换 自动 ↔ 手动 模式
切换到自动模式时自动复位所有继电器

🔄 自动 / 手动模式

🤖 自动模式（默认）

  • 传感器持续监测光照和温度
  • 光照 ADC 超过阈值 → 自动开灯；低于阈值 → 自动关灯
  • 温度超过 31.0℃ → 自动开风扇；低于 30.0℃ → 自动关风扇
  • KEY2 短按 无效（自动模式下不响应短按）

🖐 手动模式

  • 传感器自动控制停止
  • KEY2 短按循环切换继电器状态
  • PC 上位机可远程控制继电器（set_relay 指令）
  • KEY1 关闭时仍会强制切回自动模式

⌨️ 串口命令速查

命令类别功能
sWiFi扫描可用网络
*MQTT一键切换内外网（外网 8081 ↔ 内网 1883）
mqtt showMQTT显示当前 MQTT 配置
mqtt ip <IP>MQTT设置 MQTT 服务器 IP
mqtt port <端口>MQTT设置 MQTT 端口
mqtt user <用户名>MQTT设置 MQTT 用户名
mqtt pass <密码>MQTT设置 MQTT 密码
mqtt id <设备ID>MQTT设置设备 ID
mqtt saveMQTT保存所有配置到 Flash
mqtt connectMQTT立即重连 MQTT

📐 默认阈值与修改

参数默认值修改方式
风扇开启温度31.0 ℃MQTT set_threshold 指令 / PC 上位机
风扇关闭温度30.0 ℃自动 = 开启温度 - 1℃
光照开启阈值200 luxMQTT set_threshold 指令 / PC 上位机
光照关闭阈值133 lux自动 = 开启阈值 × 2/3
阈值修改后自动保存到 Flash，断电不丢失。设备重启后恢复上次设定的阈值。
